# Security Policy

This repository is an implementation scaffold for LNWP v5.0 protocol tooling.

Please do not report sensitive security issues in public GitHub issues. Use a private disclosure channel in the repository owner profile, or open a minimal issue requesting contact if no private channel is listed.

Production FULL and ENTERPRISE use still requires validated PQC, ML-DSA, TLS, JWT, mTLS, QUIC, and operational integrations. The included API server is intended for local development, tooling, and integration testing.
